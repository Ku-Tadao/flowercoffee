
# FlowerCoffee

FlowerCoffee is a simple preorder website concept for a combined flower shop and coffee shop. The project focuses on providing customers with a quick and friendly way to preorder coffee drinks and select from a small collection of flower bouquets.

## Features

- Preorder system for coffee drinks
- Menu includes espresso, americano, cappuccino and latte
- Preorder system for simple bouquets
- Flowers require a preorder at least one day in advance
- Choose pickup date and time for both products
- Clean and minimal interface intended to match the calm style of a flower and coffee shop
- Designed to be fast and mobile friendly

## Project Goals

The website aims to make the ordering flow simple and reliable. Coffee orders should be fast and suitable for same day pickup. Flower orders should be easy to understand and require minimal choices so that preparation stays manageable for the shop.
